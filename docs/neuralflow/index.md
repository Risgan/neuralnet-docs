# NeuralFlow

NeuralFlow es una aplicación móvil enfocada en la gestión de tareas personales y la planificación de actividades diarias. Está pensada para ayudar al usuario a organizar responsabilidades, controlar pendientes y recibir recordatorios según la programación definida.

## Qué hace

- Permite crear, editar y completar tareas diarias
- Soporta tareas únicas y recurrentes
- Maneja tareas con hora específica o sin horario fijo
- Muestra carga diaria y recordatorios de actividades

## Tecnologías a usar

| Capa | Tecnología |
|---|---|
| Plataforma | Android |
| Lenguaje | Kotlin |
| Arquitectura | MVVM |
| Estado y ciclo de vida | ViewModel, LiveData o Flow |
| Persistencia local | Room Database (SQLite) |
| Notificaciones y tareas programadas | WorkManager y AlarmManager |

## Enfoque del producto

NeuralFlow prioriza una experiencia simple y flexible para organizar tareas recurrentes y actividades del día a día desde el móvil.
