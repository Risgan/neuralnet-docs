# MinIO — Overview

MinIO es el servicio de almacenamiento de objetos del ecosistema NeuralNet, compatible con la API de Amazon S3.

## Responsabilidades

- Almacenamiento de archivos adjuntos y documentos
- Backups de bases de datos
- Almacenamiento de artefactos de procesos (NeuralOps)
- Archivos de carga y evidencias (TruckManager)
