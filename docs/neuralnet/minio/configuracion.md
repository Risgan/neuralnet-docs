# MinIO — Configuración

## Crear un Bucket

Desde la consola web (`http://localhost:9001`):

1. **Buckets** → **Create Bucket**
2. Nombre: ej. `neuralops-docs`, `truckmanager-files`
3. Configurar permisos según visibilidad requerida

## Crear Access Keys

1. **Access Keys** → **Create access key**
2. Guardar `Access Key` y `Secret Key` en Vault

## Política de acceso por aplicación

Crear una política específica por sistema para limitar acceso solo a sus buckets correspondientes.
