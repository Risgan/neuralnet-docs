# NeuralNet

NeuralNet es la marca principal bajo la que se organizan los aplicativos y servicios del ecosistema. Dentro de NeuralNet se define una base tecnológica común con componentes reutilizables que pueden ser compartidos por NeuralOps, TruckManager, NeuralFlow y futuros proyectos.

## Qué hace

- Define una base tecnológica común para todos los aplicativos
- Centraliza componentes reutilizables de infraestructura y seguridad
- Permite acelerar nuevos proyectos reutilizando servicios ya resueltos
- Facilita despliegue, integración y escalabilidad desde una misma plataforma

## Tecnologías a usar

| Componente | Tecnología |
|---|---|
| Contenedorización | Docker y Dockerfile |
| Identidad y acceso | Keycloak |
| Base de datos | PostgreSQL |
| Almacenamiento de archivos | MinIO |
| Gestión de secretos | Vault |
| Mensajería de eventos | Kafka |
| Caché y sesiones | Redis |

## Rol dentro del ecosistema

NeuralNet no se presenta como un aplicativo aislado. Su función es representar la marca y alojar la plataforma compartida donde viven los componentes reutilizables del ecosistema.
