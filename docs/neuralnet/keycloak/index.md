# Keycloak — Overview

Keycloak provee el servicio centralizado de autenticación y autorización (SSO) para todos los sistemas del ecosistema NeuralNet.

## Responsabilidades

- Gestión de usuarios y roles
- Single Sign-On (SSO) entre NeuralOps, TruckManager y NeuralFlow
- Emisión de tokens JWT (OAuth2 / OIDC)
- Integración con directorios externos (LDAP/AD)
