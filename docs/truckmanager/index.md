# TruckManager

TruckManager es un sistema orientado a la gestión y control de operaciones de transporte de carga. Permite administrar flota, conductores, rutas, viajes y cargas, dando visibilidad operativa sobre todo el ciclo logístico.

## Qué hace

- Gestiona vehículos y conductores
- Controla viajes y operaciones logísticas
- Registra cargas y documentación asociada
- Genera reportes operativos para seguimiento del negocio

## Tecnologías a usar

| Capa | Tecnología |
|---|---|
| Frontend web | React |
| Backend | .NET |
| Aplicación móvil | Kotlin |
| Infraestructura compartida | Servicios de NeuralNet |

## Enfoque

TruckManager aprovecha la plataforma común para integrarse con autenticación, almacenamiento y otros servicios del ecosistema cuando sea necesario.
